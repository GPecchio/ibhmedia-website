# To run do the following
git clone https://github.com/ibh-media/ibhmedia-website.git

`cd ibhmedia-website`

### On Windows:
`ibhenv\Scripts\activate`
### On Mac/Linux:
`source ibhenv\Scripts\activate`

`pip install -r requirements.txt`

`python manage.py runserver`
